### 作品

这些作家的作品都是什么流派的？

（分析流派随着时间发生的变化）

悬疑作品流派：（阿加莎以前很少进行分类）

- **本格推理：阿加莎为主**
- 社会推理：日本作家【东野圭吾】
- 变格推理：日本作家【脑髓地狱，金田一探案集】
- 新本格推理：日本作家  

本格推理是最早出现的推理小说种类，其代表人物与作品大多是欧美，但是随着时间变化，越来越多的新流派，新潮流转向了**日本作家**【*】
